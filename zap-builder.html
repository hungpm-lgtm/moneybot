🎯 Cách C — Widget iOS hỏi tiền mỗi lần (ĐỀ XUẤT)

Tạo 1 widget trên homescreen. Bấm widget → iPhone hỏi "Bao nhiêu tiền?" → bạn nhập → tự lưu. Số tiền không cố định — phù hợp danh mục bạn chi tiền khác nhau mỗi lần.

Chọn danh mục trước...

📲 Hướng dẫn cài Shortcut + Widget:

Bước 1. Mở app "Phím tắt" (Shortcuts) — icon ô vuông tím Apple cài sẵn
Bước 2. Bấm tab "Phím tắt của tôi" → bấm + góc trên phải
Bước 3. Bấm "Thêm Hành Động" (Add Action) → ô tìm kiếm gõ "yêu cầu nhập" (hoặc tiếng Anh: "Ask for Input") → chọn nó
Bước 4. Ở hành động vừa thêm:
• Trường "Câu hỏi": nhập "Bao nhiêu tiền?"
• Bấm vào "Văn bản" ở giữa → đổi thành "Số" (Number)
Bước 5. Bấm "Thêm Hành Động" lần nữa → gõ "mở URL" (hoặc "Open URL") → chọn
Bước 6. Ô URL: sao chép URL base ở trên rồi dán vào. URL kết thúc bằng &a=
Bấm vào CUỐI URL (sau dấu =) → trên bàn phím sẽ có thanh gợi ý chứa "Provided Input" (hộp xanh) → bấm để chèn
Bước 7. Bấm tên Shortcut ở trên cùng (mặc định "Phím tắt mới") → đặt tên ngắn, VD: "🍽️ Ăn" → chọn màu + biểu tượng
Bước 8. Bấm "Xong" ✓ — Shortcut đã tạo xong. Thử bấm vào nó để test
Bước 9 — Đặt làm widget trên homescreen:
• Ra màn hình chính → giữ vùng trống → góc trên trái bấm +
• Tìm "Phím tắt" (Shortcuts) → chọn widget cỡ nhỏ (1×1)
• Bấm "Thêm Widget" → bấm "Xong"
Bước 10. Bấm giữ widget vừa thêm → "Chỉnh sửa Widget" → chọn Shortcut "🍽️ Ăn" vừa tạo → bấm chỗ khác để xong
✅ XONG! Giờ trên màn hình chính có widget "🍽️ Ăn".
• Bấm widget → iPhone hỏi tiền → nhập số (VD: 75000) → OK
• Tự mở web → lưu 75.000₫ vào Thức ăn → toast "✓ Đã lưu" → đóng tab

Mẹo: Lặp lại cho mỗi danh mục hay dùng (Ăn / Xăng / Cafe / Mua sắm...). Có thể đặt 4 widget cùng 1 ô lớn (cỡ 2×2) bằng tính năng Smart Stack của iOS.

📌 Cách A — Bookmark cố định (1 tap)

Số tiền cố định luôn — phù hợp combo lặp đi lặp lại y hệt (VD: Cafe 30k mỗi sáng, Netflix 851k mỗi tháng).

Chọn danh mục và số tiền để tạo URL...

Cách add bookmark iOS:

1. Sao chép URL ở trên
2. Mở Safari → dán URL vào thanh địa chỉ → mở
3. Bấm nút chia sẻ ⬆️ ở dưới → "Thêm vào Màn hình chính"
4. Đặt tên ngắn (VD: "☕ Cafe 30k") → Thêm
✅ Bấm icon trên homescreen = lưu ngay không hỏi gì
