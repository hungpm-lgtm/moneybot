3. Cách add vào màn hình chính

Cách 1 — Bookmark đơn giản (1 tap):

1. Sao chép URL ở trên
2. Mở Safari → dán URL → mở
3. Bấm nút chia sẻ ⬆️ → "Thêm vào Màn hình chính"
4. Đặt tên ngắn gọn (VD: "Ăn 50k") → Thêm
✅ Từ giờ bấm icon trên màn hình = lưu giao dịch ngay

Cách 2 — Widget thật (cần Shortcuts.app):

1. Mở app "Phím tắt" (Shortcuts) của Apple
2. Tab "Phím tắt của tôi" → + → Thêm hành động
3. Tìm "Mở URL" → dán URL ở trên
4. Bấm tên Shortcut → đặt tên (VD: "Ăn 50k") → chọn icon ☕🍽️
5. Ra màn hình chính → giữ → + → tìm "Phím tắt" → chọn widget → Add Widget
6. Bấm vào widget để chọn Shortcut vừa tạo
✅ Giờ có widget thật trên homescreen

Cách 1 — Long-press icon app (đề xuất):

Sau khi cài MoneyBot làm PWA → giữ icon MoneyBot → hiện menu với các lối tắt sẵn (Ăn 50k, Cafe 30k...).

Cách 2 — Bookmark từng URL:

1. Sao chép URL ở trên
2. Mở Chrome → dán URL → mở
3. Menu 3 chấm ⋮ → "Thêm vào màn hình chính"
4. Đặt tên ngắn gọn → Thêm
✅ Bấm icon trên màn hình = lưu ngay
